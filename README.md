# Sanchia Recson Lakkarvi - Portfolio Website

Professional static portfolio website for Sanchia Recson Lakkarvi, showcasing AI, data analytics, automation, healthcare data, Power BI, Python, SQL, AWS, projects, publications, awards and UWA leadership.

## What is included

- `index.html` - main portfolio website
- `resume.html` - public resume page
- `assets/docs/Sanchia_Recson_Lakkarvi_Public_Resume.pdf` - public-safe resume PDF
- `assets/css/style.css` - website styling
- `assets/js/main.js` - filters, mobile menu, theme toggle and gallery behavior
- `assets/img/` - approved portfolio images, certificates and profile photo

## Important upload instruction

When uploading to GitHub Pages, upload the files **inside** this folder, not the outer folder itself.

Correct GitHub root should show:

- `index.html`
- `resume.html`
- `assets/`
- `README.md`

Wrong upload example:

- `sanchia-portfolio/index.html`

If GitHub shows `sanchia-portfolio/index.html`, the website will not open correctly at the main link.

## Publish on GitHub Pages

1. Create a repository named `SanchiaLakkarvi.github.io`.
2. Upload all files inside this folder.
3. Commit changes.
4. Go to Settings -> Pages.
5. Select Deploy from branch, branch `main`, folder `/root`.
6. Open: `https://sanchialakkarvi.github.io/`.
